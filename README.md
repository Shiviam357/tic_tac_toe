# Tic-Tac-Toe Python (V1)

A simple, terminal-based Tic-Tac-Toe game built with Python.

## Features
- Interactive 3x3 grid using numbers 1-9.
- Turn-based gameplay for two players (X and O).
- Basic win detection and tie handling.
- Input validation to prevent overwriting moves.

## How to Run
1. Ensure you have Python installed.
2. Download `tictactoe.py`.
3. Run the command: `python tictactoe.py`
